## 4. 공공주택 용어집 (Glossary)

### 기본 정보
- **파일 위치**: `lab/오흥재/Glossary_1000/`
- **파일명**: `glossary_FINAL_1000.json`
- **용어 수**: 1,000개 이상
- **형식**: JSON

### 구성 요소
각 용어 객체는 다음 필드를 포함합니다:
- `id`: 용어 고유 ID (예: CI_001)
- `term`: 용어명
- `major_category`: 대분류 (계약·입주, 비용·세금·대출 등)
- `category`: 세부 카테고리
- `definition_easy`: 쉬운 설명
- `definition_full`: 상세 설명
- `usage`: 사용 예시
- `example`: 실제 사용 예시
- `related_terms`: 관련 용어 리스트
- `tags`: 태그 리스트
- `source`: 출처 (LH, SH, GH 등)

### 대분류별 분포
- 계약·입주: 239개 (23.8%)
- 비용·세금·대출: 196개 (19.5%)
- 관리·유지보수: 143개 (14.2%)
- 서류·시스템: 132개 (13.1%)
- 부동산_기초: 130개 (12.9%)
- 법률·권리: 103개 (10.2%)
- 공고·절차: 62개 (6.2%)

### 활용 목적
- RAG 시스템에서 전문 용어 설명 제공
- 사용자 질문에 포함된 용어 자동 인식 및 설명
- LLM 답변 생성 시 용어집 정보 활용
